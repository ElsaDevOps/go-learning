# Word Counter

The WordCount function returns the map of the counts of each word in the string s. The wc.Test function runs a test suite against the function and prints success or faliure.

## What I learned

- How to count words in a string using a for loop.
- How to test that a key ispresent in a map with a two value assignment.
- how to use the function Fields to split up a string.
