# Learning Go for Platform Engineering

A 4-week learning path focused on building production-grade infrastructure tools with Go.

## About

Learning Go to contribute to infrastructure projects (Terraform, Kubernetes, Prometheus) and build platform engineering tools.

**Main Project:** Production-ready URL shortener with Kubernetes deployment  
**Timeline:** December 2025 - January 2026

## Repository Structure
```
week-01-fundamentals/    # Syntax, functions, error handling
week-02-http-servers/    # REST APIs, routing
week-03-databases/       # PostgreSQL + GORM, URL shortener
week-04-production/      # Docker, Kubernetes deployment
```

## Tech Stack

Go 1.21+, PostgreSQL, GORM, gorilla/mux, Docker, Kubernetes
